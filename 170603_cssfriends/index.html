単位の話を使い方などをダラダラ話ます。

px

絶対値

大体の人が最初に覚える単位ですよね？

em

相対値

親、祖先要素のフォントサイズを基準とした単位。

rem

相対値

ルート要素のフォントサイズを基準とした単位。

          
            /* ルート要素（html タグ）を 10px に指定しておいて */
            html {
              font-size: 62.5%;
            }
            /* p 要素を 16px に指定するには以下の様に記述する  */
            p {
                font-size: 1.6rem;
            }

vw、vh

vw: viewport width ビューポートの幅に対する割合

vh: viewport height ビューポートの高さに対する割合

width に 100vw と 100% の指定の違いは、vw はブラウザのウィンドウにスクロールバーが表示されているとそこも含まれているので注意が必要です。

Mac の設定でスクロールバーを隠す設定していると気が付かないことがあるので、覚えておくといいと想います。

単位まだまだあります。。

ブラウザの拡大・縮小の機能が効きません。

文字サイズを指定は em や % や rem を使うといいと思います。

            .square {
              width: 50vw;
              height: 50vw;
            }
